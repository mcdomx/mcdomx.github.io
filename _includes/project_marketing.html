Pathfinder

Off-road Navigation

Using Keras and OpenCV, Pathfinder will highlight a navigable path on a video of off-road terrain. The model was trained on a small set of publically available data that was augmented to produce a useful set of training data.

RaspPi-Cam Data Collection

Platform for visual data collection

This project uses a series of Raspberry Pi cameras in conjunction with Docker containers and ROS. Cameras are networked and collect image data. Images and identified and statictical data collected.
